== Description ==

Add a widget that displays your bcna address in the site footer.

== Installation ==

This section describes how to install the plugin and get it working.

Once the plugin is activated, you just have to go to the wiget part in the administration panel and drag the widget into the active part of the site.
Add a name to your widget and your bitcanna address.

You will see a block appear with your donation address on the main page of your site
